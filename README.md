## Proje Hakkında
Bu proje, bir banka yönetim sistemini simüle eden Java uygulamasıdır. Müşteri oluşturma, hesap açma, para transferi ve kredi kartı yönetimi gibi işlemleri içerir.

## Paket Yapısı
| Paket | İçerdiği Sınıflar |
|-------|-------------------|
| com.bank.app.people | Kisi, Musteri, BankaPersoneli |
| com.bank.app.accounts | BankaHesabi, VadesizHesap, YatirimHesabi |
| com.bank.app.cards | KrediKarti |
| com.bank.app.service | BankaService |
| com.bank.app.main | Main |

## Yapılan İşlemler
- Müşteri oluşturma
- Vadesiz ve Yatırım hesabı açma
- Kredi kartı tanımlama
- Para transferi
- Kredi kartı borcu ödeme
- Hesap ve kart silme işlemleri
